measuring.png

instructions:
open all three files
open monster
lay cutebound helmet over cutebound head
copypaste onto monster
select up to outer measuring border
put layers into folder
turn both layers to image material
tile both
rasterize
move around
match head color to rest of monster
hide folder
magic wand skin color
layer mask on folder
reduce opacity
use lasso tool to draw on layer mask
check for impurities
make new folder if needed
copy path of new folder
shift+S to save as
paste path, name
alt+w to save png duplicate
